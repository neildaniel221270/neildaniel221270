landmarks_detection Module¶

landmarks_detection.detect_landmarks(mask)¶

Detect landmarks such as primary root start and end points, and lateral root tips from the provided mask.

@param mask: Binary mask representing the segmented instances. @type mask: numpy.ndarray @return: List of dictionaries containing the coordinates of primary root start and end points,

and lateral root tips.
