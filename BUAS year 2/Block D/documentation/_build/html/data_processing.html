Preprocess images and their corresponding masks by cropping, padding, resizing, and patchifying them. Saves the processed patches to the specified output directory.

@param input_dataset_path: Path to the input dataset directory. @type input_dataset_path: str @param output_dataset_path: Path to the output dataset directory. @type output_dataset_path: str @param scaling_factor: Factor by which to scale the images and masks, defaults to 1. @type scaling_factor: int, optional
